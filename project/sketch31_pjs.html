Un exemple de Dave Briccetti que vous pouvez retrouver dans ce dépôt Github, accompagné de plusieurs autres exemples intéressants.

J'ai retravaillé la version de Dave pour séparer au maximum la mécanique de gestion de l'horloge, de la mécanique propre à P5. Cela m'a permis de porter ensuite plus facilement le code vers ProcessingJS. La conversion vers ProcessingJS était particulièrement intéressante du fait que ProcessingJS n'a pas de primitive 3D de type cylindre. Cela m'a obligé à employer une fonction "cylinder" directement inspirée du sketch "cylindres" (sketch11) qui se trouve dans le présent projet. Ma fonction "cylinder" n'étant pas parfaitement identique, j'ai dû jongler avec les rotations pour obtenir le même rendu que dans la version P5. C'était un exercice très intéressant.
